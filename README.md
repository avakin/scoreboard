# Simple Live score board tracker

Plain VIte+React template which can be run with following commands:

```
npm i && npm run dev
```

or

```
yarn && yarn dev
```

# Testing

To run test cases :

```
npm run test
```

and to see coverage:

```
npm run coverage
```
